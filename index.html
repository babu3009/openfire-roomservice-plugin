openfire-roomservice-plugin

Openfire roomservice plugin to allows administration of rooms and MUC service via HTTP requests.

Configuration page included into plugin like roomservice plugin. The service address is [hostname]/plugins/roomservice/roomService Parameters:

  • jid
  • roomName
  • secret
  • subdomain
  • type
  • description

Supported type: 'add', 'delete', 'createMultiUserChatService' and 'removeMultiUserChatService' (updateMultiUserChatService -> hidden)

Reference http://www.igniterealtime.org/projects/openfire/plugins/roomservice/readme.html

Remember to configure plugin in Openfire admin console.

Example

http://10.10.1.32:9090/plugins/roomService/roomservice?type=createMultiUserChatService&secret=pWLz65KU&subdomain=demo1&description=this+is+a+demo+testing
